The App Concept: "AutoPortfolio" (or "Kinetic")
The Elevator Pitch: An AI-powered "black box" for your professional life that automatically captures your daily work achievements, cleans up the messy context, and translates it into a beautiful, verified "Proof of Skill" feed without you ever having to sit down and write a resume.
The Hackathon Prompt
"Build 'Kinetic,' a productivity and career-growth app that acts as an automated ghostwriter for a user's professional 'Proof of Skill.'
The app allows users to quickly dump raw, messy context throughout the day—such as screenshots of working code, voice memos summarizing a breakthrough, Git commit logs, or rough project notes. Using generative AI, the app automatically parses this unstructured 'life admin' data to do two things:
Organize the Admin: Extract action items, log project milestones, and update a personal progress dashboard.
Generate the Proof: Ghostwrite polished, professional 'Proof of Work' snippets (complete with technical tags, metrics, and visual cards) that document what the user actually built or achieved.
The app features a high-speed, TikTok-style visual feed of these verified achievements that users can instantly share with teams, clients, or recruiters, turning daily chaos into a continuous, living portfolio."
Why This is a "Killer" App (The Viral Flywheel)
Solves "Documentation Fatigue": Everyone hates updating resumes, LinkedIn, or portfolios. By moving the friction from "sit down and write an essay about your skills" to "just drop a screenshot or a 10-second voice note," you create a massive daily utility habit.
The High-Retention Feedback Loop: The app provides immediate value. You input a messy brain-dump, and the AI hands you back an organized calendar/to-do list and a beautiful piece of career collateral.
Built-in Growth Loop: When users want to show a boss why they deserve a promotion, or show a client what they did this week, they will send a link to their "Kinetic Proof Feed." Every shared link is a high-conversion landing page for new users.
The "Magic Demo" Recipe for the Judges
To win the hackathon, your live demo needs to show this transformation instantly.
Step 1 (The Input): Show a messy input screen. Type something chaotic or upload an ugly image. Example: A screenshot of a messy terminal error that was finally fixed, accompanied by a voice note: "Finally got the webhook working after fighting with the API headers for three hours, ready for staging."
Step 2 (The Magic Mirror): Hit process. Show the AI splitting this into two clean outputs:
The Admin Side: A clean task marked as "Done" and a prompt to draft a Slack update for the team.
The Proof Side: A sleek, dark-mode visual card titled: "API Integration & Webhook Architecture." It lists the specific technologies used, calculates time-to-resolution, and writes a crisp, data-backed summary of the technical achievement.
Step 3 (The Share): Show a one-click generation of a public portfolio link containing that card.
Suggested Tech Stack for Speed
Frontend: Flutter or React Native (for seamless mobile photo/voice capture).
Backend/Database: Supabase (for rapid user auth, quick schema setup, and storage of media files).
AI Engine: Gemini API (Structured Outputs feature to reliably split the raw input into admin_tasks and portfolio_proof JSON objects).
